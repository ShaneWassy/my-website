Mobile App

Download App

In my CS481 Mobile Programming course, I created a cook book app for IOS. The purpose of the app is to have a place to store instructions and ingrediants for favorite recipes.

I designed this app for personal use as well as to learn more about the react native and xcode environments. Using html and css, I designed each page of the app. Using react native and xcode, I built a launchable app that can be used.

I learned valuable information around mobile app development while building my app. Using figma to design mock-ups and wireframes to understand what is needed for user's before coding.

How to run:

    In order to run the app, user must have xcode with IOS 18.0+ simulators and up to date react native environment.

    Once all softwares have been downloaded and installed...

    1. Open terminal, cd AwesomeProjectP

    2. In AwesomeProjectP directory, npm start

    3. Once react native is launched, i (for IOS), simulator will launch and app will open

    If any errors occur or failure to launch app...

    1. Type "npm install"

    2. Type "cd ios", "pod install", "cd .."

    3. Run again
