2.5D Game

Download Game File

Using a maze layout on a cartesian coordinate system, I placed down 4 images on each point of a 5x5 grid to make a scrollable 2.5D maze. Using the arrow keys you can navigate through the images and move locations.

How to Play:

    1. Download the game from above

    2. Unzip the downloaded file

    3. Open terminal, cd 2.5D, type "python3 -m http.server"

    4. Open browser, search "http://localhost:8000/canvas.html"
